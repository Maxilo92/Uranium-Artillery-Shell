# Uranium Artillery Shell

**Uranium Artillery Shell** is a Factorio mod that adds a powerful new ammunition type for your artillery turrets and wagons.

## Features

*   **New Item**: Uranium Artillery Shell.
*   **Enhanced Damage**: Deals significantly more physical and explosion damage than standard shells.
*   **Larger Radius**: Explosion radius is doubled (8.0 vs 4.0).
*   **Radiation Cloud**: Leaves behind a lingering cloud of radioactive fallout that damages enemies over time (30 seconds duration).
*   **Visuals**: Green-tinted shell icon and green radiation cloud.

## Recipe

*   **Crafting Time**: 20 seconds
*   **Ingredients**:
    *   1x Artillery Shell
    *   1x Used Up Uranium Fuel Rod
    *   5x Uranium-238

## Technology

Unlocks with the **Uranium Artillery Shell** research.
*   **Prerequisites**: Artillery, Nuclear Power.
*   **Cost**: 2000 of Automation, Logistic, Chemical, Military, and Utility science packs.

## Installation

1.  Download the latest release or clone this repository into your Factorio `mods` directory.
2.  Ensure the folder name matches the mod name and version (e.g., `Uranium-Artillery-Shell_0.0.1`).
3.  Enable the mod in the game menu.

## Compatibility

*   **Factorio Version**: >= 2.0.0
*   **DLC**: Works with or without Space Age DLC.
